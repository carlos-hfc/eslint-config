# ESLint Config

ESLint configuration for React and Node.js projects.

## Setup

### React 

Install dependencies:

```bash
npm i -D eslint @ch-faustino/eslint-config
```

Create eslint file:

```bash
touch .eslintrc.json
```

Add to `.eslintrc.json`:

```json
{
  "extends": "@ch-faustino/eslint-config/react"
}
```

### Node.js

Install dependencies:

```bash
npm i -D eslint @ch-faustino/eslint-config
```

Create eslint file:

```bash
touch .eslintrc.json
```

Add to `.eslintrc.json`:

```json
{

```
